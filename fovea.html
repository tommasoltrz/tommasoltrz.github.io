Chrome Extension: January 2017
FOVEA

In a nutshell

Personalised search results, news-streams and advertisements can lead us in a state of intellectual isolation.

If you like this, you will like that.

Fovea is a Chrome Extension, that helps breaking up the filter bubbles. It allows the user to manipulate the Google algorithm by forcing it to show first a certain kind of results, based on a database edited by the user itslef.

Developed at Malmo university, Fovea was born as a critical design project to than be turned into a working tool.

Features

The Fovea default database includes some of the main news websites's names sorted by political inclination.
By dragging the slider left, the more democratic results will be shown first, while dragging it to the right the conservative ones will be prioritised.

The UI equality switch reduce to the minimum every possible distraction by making the appereance of a website as neutral as possible. It creates an iframe that display only plain text formatted in the most neutral way.

The Bubble switch inspects the browsing history of the user and block every website that has already been visited.
